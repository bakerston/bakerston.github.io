# 武将介绍

# 魏国

# 曹操

  • 曹操在最初的老版1v1是四禁将的强度。除去额外的一体力加成外，主要的原因是曹操是不会缺少输出的：对手造成的伤害牌都会被枭雄后成为他的输出牌。加上老版1v1使用的是标风牌库，输出牌占比并不高，缺乏火力，因此曹老板的进攻属性非常强。
  • 新版1v1增加了输出牌的比例，平均下来每回合都会拿到一输出牌，在这种几乎不会缺少伤害的情况下，曹老板再拿到额外输出牌的作用被稀释了。目前曹老板的优势主要在于【换命】，在需要强行带走对面的时候出场会有稳定的发会，但在四平八稳的局里能力一般，也容易被对面核心养猪。
  • 适合后手一号位出场。

TIP

攻击：60 防御：30 控制：50 综合：B+

# 徐晃

  • 改版前有着全模式里一人之下万人之上的强度，最恶心的武将没有之一，有着让对手双手离开键盘任人鱼肉的技能，也是面杀局中默认要禁掉的武将，徐晃+会玩的玩家会带来超过30分钟一盘的单挑局。

WARNING

【养】这个打法形容徐晃再合适不过，这是一种缓慢但追求收益极限的打法。如果使用的武将是吕布，或者马超等普通菜刀，玩家希望的是尽快带走对方，武将的状态并不重要，达到一换一就是他们的期望强度了。

但徐晃并不一样，徐晃的上限极高，与对面的菜刀武将一换一是非常大的浪费，徐晃的期望强度更可能是一打二或者一打三。因此，如何在杀掉当前对手武将的同时，保证自身状态能够顺利面对下一位武将，这是徐晃玩家需要考虑的战术，也就是【养】。

具体一点：如果你当前的血量是2血2牌，面对新登场的庞德，你有多大的胜算打赢？ 如果你有4血6牌2装备，手中有无中生有、过河拆桥、无懈可击、乐不思蜀，你又有多大的胜算打赢？徐晃的能力，就是能把自己从第一个状态养到第二个状态的同时让对面的武将苟延残喘，等到养起足够的状态，带走当前残血武将后以非常充足的手牌面对下一位武将。

  • 一张【兵粮寸断】的牌差收益是0.5，看似微不足道，但徐晃就是靠着这每回合半张牌积少成多，慢慢地积累起满状态，带走对面的同时还有足够的牌应付下一个武将。单看一个结果还远不能体现游戏过程的折磨程度，如果对面连续过了几个【兵粮寸断】，破坏了徐晃的手牌结构，会玩的徐晃玩家就会重新养起来。这个过程对于竞技双方都是不小的折磨。

  • 打法非常简单，除了防具慎重使用外，基本上可以无脑当作【兵粮寸断】使用。

WARNING

有意思的是，如果只分析牌差，其实是有不少武将从技能机制上“克制”徐晃：

  • 曹仁：徐晃两回合过4摸3，而曹仁每两回合过3摸3，考虑到【兵粮寸断】25%的几率不中，曹仁每两回合有3.5牌的摸牌量，而徐晃2回合兵一次，每两回合有4-1=3牌的摸牌量，曹仁每回合领先0.25牌。
  • 郭嘉：这个不用说了，天妒直接拿到判定牌，跟徐晃一样每回合摸一牌（只不过这张牌是明置的），考虑到水兵的概率，郭嘉的平均摸牌量1.25同样大于徐晃1。
  • 姜维：徐晃的黑色杀都被拿来当作【兵粮寸断】，几乎不会有额外的杀来防【挑衅】，虽然姜维没有额外的摸牌，但可以把徐晃蓄爆的牌拆个七七八八，理想情况下姜维0.25，徐晃接近0。
  • 貂蝉：每回合稳定一张【闭月】，貂蝉1.25，徐晃1。
  • 甄姬：每回合平均一张【洛神】，甄姬1.25，徐晃1。
  • 夏侯渊：夏侯渊在不得已时，可以

但为什么这些武将打徐晃的时候仍然很吃力？因为摸牌是一回事，过牌量是另外一回事，哪怕【兵粮寸断】有概率不中，哪怕对手有一些额外的摸牌，徐晃的过牌量依然是数倍于对手，摸到收益牌的概率也会更高。

TIP

攻击：60 防御：90 控制: 95 综合：S

改版后，徐晃【兵粮寸断】只能在打出攻击牌后使用，他也不再具备蓄爆能力，失去了大核心的地位，但对付普通的菜刀还是绰绰有余。

# 司马懿

  • 虽然有不错的防御威慑，掉血后能拉卡牌差，但游戏的主动权在对面。如果对面有能力选择控场发育，司马懿跟其他白板一样没有还手能力。
  • 而且，说是防御和牌差能力不错，但期望的收益也就是死前的两次反馈，考虑到对手可以选择先用掉收益牌后留一些废牌在手中等待反馈，司马懿的反馈收益通常也不会很理想。
  • 一般情况下没有出场空间，更适合后手一号位出场。

TIP

攻击：30 防御：50 综合：B-
