# 为什么有“上单吃等级，AD吃装备”的说法？

（这是我20年在虎扑上发的贴子，虽然版本更迭日新月异，但这些基础逻辑还是没有变化的）

开门见山： 因为上单英雄造成的伤害中，由基础属性和成长属性造成的伤害占比较高； 而在ADC所造成的伤害中，该比例较低，来自装备的伤害占比较高。

（这里仅比较常见的上单与ADC，至于射手位上单、坦克下路等冷门选择并不在讨论范围内。实际上这些英雄还是应该以他们的机制来划分定位，而绝非他们走的是哪条路，盖伦去了下路，依然是按照上单的属性来分析，卢锡安走了上路，也要归于adc一类）

所谓上单的伤害依赖基础属性的理由有二：

# 第一个因素：上单英雄的基础属性更高。

每个等级之间的属性会差很多，但ADC基础属性远不如上单，因此等级造成的属性差不如上单明显。

举个栗子： 上路鳄鱼vs诺手，诺手每加一级，攻击力多5点，护甲多4点，血量多100点。 下路小炮vs寒冰，小炮每加一级，攻击力多3.3点，护甲多3点，血量加88点。

对比还是比较明显的，这代表着什么？ 代表着大部分上单英雄，可以仅靠基础属性，就打出足够的伤害；而同样的思路，放在ADC身上是行不通的，ADC必须买大量的攻击力加成装备，才能打出够看的输出。

我们把英雄联盟所有英雄，按照攻击力（AD）排序，可以注意到排名靠前的都是常见的战士英雄。

相反，我们能注意到很多ADC分布在列表末尾。

不止攻击力，在体力值、护甲、基础生命回复等属性上，ADC的基础属性远不如上单英雄。

# 第二个重要的因素是，两个位置打输出的模式不同。

上单英雄更多地是技能衔接普攻，极度依赖技能的基础伤害加成; ADC的伤害来源中，普攻的占比更高，技能伤害的影响不明显。

大家可以代入自己的日常对线：诺手打鳄鱼，诺手释放的QWR和平A、鳄鱼的QWE和平A，每个环节都是依赖AD加成的，所以整体而言升一级加区区几点ad，看似无所谓，但对于他们的一套连技能带平A的组合拳来说，增加的伤害就有两位数了，不容小觑。

但是放到下路，两位ADC的对拼，伤害来源更多地来自他们的平A，对于技能的依赖不如上单。比如，女警玩家并不会以她的Q为主要伤害来源，卢锡安的Q，寒冰W，金克斯的W均是同理。这些AD英雄并不是没有伤害型技能，只是这些技能远远算不上他们的主要伤害来源。所以差一级的技能，对于AD而言并不如上路那般致命。

当然了，这不代表上单就只需要等级而不需要装备，不代表ADC只需要装备而不吃等级。毕竟，哪个诺手鳄鱼不希望自己白多出个幽梦无尽？哪个adc不希望自己早早18级？ 只是相对而言，他们的强项受更重要的因素支配。好钢用在刀刃上，上单作为队伍的前排，需要承担部分坦克的责任，因为只能（含泪）做肉装，所以他们的输出更加依赖于基础属性，而ADC既然基础属性极低，自然是放弃出坦克装备，而是把所有的经济用在输出装备上，所以他们的伤害j更多地取决于经济和装备了。
